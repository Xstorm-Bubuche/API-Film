# CinéAPI

Une API REST PHP simple pour consulter des films et gérer une liste de favoris.

Cette API fournit des informations sur des films selon différents critères (populaires, mieux notés, à l’affiche…) et permet à un client de rechercher des films ainsi que de gérer une liste de favoris.

# Installation

1. Cloner le dépôt

2. Configurer l’environnement

- Assure‑toi que PHP est installé (version recommandée ≥ 7.4).
- Place le projet dans un serveur web local (ex : Apache, Nginx, ou via php -S localhost:8000).

# Lancement

- L’API s’exécute depuis le fichier index.php.
- ou avec la commande "php -S localhost:8000" puis en allant sur "http://localhost:8000/public/index.html"
